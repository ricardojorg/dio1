Estrutura Básica do HTML

A estrutura básica do HTML é essencial para que sua página seja corretamente interpretada pelos navegadores.

A estrutura básica de uma página HTML consiste em duas partes que são separadas pelas tags <head> e <body> e estas devem estar dentro da tag <html >.

O arquivo principal ou seja, a primeira página, deve ser nomeado como index.html, isso faz com que os servidores de hospedagem exibam corretamente a página inicial ao receber uma requisição do client.

Falando Sobre Tags

São explicadas as funções das seguintes tags:

  • <html> - especifica o início e o fim do arquivo HTML.
  • <head> - entre essas tags são usadas outras tags específicas.
  • <body> - é a tag que especifica todo o conteúdo exibido na página.
  • <title>* - define o título da página que aparece na aba do navegador.
  • <i> - formata o texto para itálico.
  • <strong> - formata o texto para negrito.
  • <input> - utilizada para definir campos de formulário.
  • <img> - insere uma imagem no conteúdo da página.

Atributos das Tags

Os atributos de tags são configurações específicas que a tag poderá ter. Há atributos globais, isto é, podem ser utilizados em várias tags e alguns atributos são específicos de uma tag.

Textos

São explicadas as funções das seguintes tags:

  • <hn> - aonde "n" representa um número de 1 a 6, a tag <h1> é utilizada para um título de tamanho 1, o título principal, enquanto que as tags <h(2 a 6)> são para subtítulos, aonde quanto maior o número menor é o tamanho do subtítulo.
  • <blockquote> - é utilizada para definir que o texto é uma citação.
  • <u> - é utilizada para sublinhar.
  • <mark> - é utilizada para marcar o texto, um destaque em cor.
  • <sup> - utilizada para formatar o texto como sobrescrito.

Listas Ordenadas e Não Ordenadas

São listas que podem ser numeradas ou com símbolos para cada item.

Utiliza-se as seguintes tags:

  • <ol> - é utilizada para listas ordenadas.
  • <ul> - é utilizada para listas não ordenadas.
  • <li> - é utilizada para definir cada item da lista.

Links

  • <a> - define o objeto a ser lincado.
